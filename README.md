## ‚ú® Key Features

-   **immersive 3D Experience**: Integrated **Three.js** and **React Three Fiber** for real-time 3D rendering, featuring a physics-based interactive ID badge.
-   **Dynamic UI/UX**: Smooth transitions and animations powered by **Framer Motion**.
-   **Bento Grid Layout**: A modern, responsive masonry layout for showcasing projects.
-   **Interactive Map**: Custom **Leaflet** map integration to display location with style.
-   **Theme System**: Fully supported Light and Dark modes with persistent state.
-   **Performance Optimized**: Built with **Vite** for lightning-fast HMR and optimized production builds.

## üöÄ Getting Started

Follow these steps to set up the project locally on your machine.

### Prerequisites

-   Node.js (v16.0.0 or higher)
-   npm or yarn

### Installation

1.  **Clone the repository**
    ```bash
    git clone https://github.com/anothersunrisee/fajarahnf.git
    cd fajarahnf
    ```

2.  **Install dependencies**
    ```bash
    npm install
    ```

3.  **Start the development server**
    ```bash
    npm run dev
    ```
    Open [http://localhost:5173](http://localhost:5173) with your browser to see the result.

## üì¶ Building for Production

To create a production-ready build:

```bash
npm run build
```

This will generate optimized static assets in the `dist` directory.
